Welcome to Palpiction

The following interface guides you through a series of image.
For each image, touch the image area. When you touch an identified object, it will be read out. For a full list of image objects to be read out, touch the 'Hover for All Objects' area. When you are comfortable with the contents in the image, select the 'Next' button.
IMPORTANT: Please turn on a Accessibility screenreader before using (eg. VoiceOver or Talkback). Please do not refresh this page at any point in time.
